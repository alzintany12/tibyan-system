## 🚀 الخطوات التالية للتشغيل

1. **تثبيت Dependencies:**
   ```bash
   composer install
   npm install && npm run build
   ```

2. **إعداد قاعدة البيانات:**
   ```bash
   cp .env.example .env
   php artisan key:generate
   php artisan migrate
   php artisan db:seed
   ```

3. **إعداد التخزين:**
   ```bash
   php artisan storage:link
   chmod -R 775 storage bootstrap/cache
   ```

4. **تشغيل الخادم:**
   ```bash
   php artisan serve
   ```
